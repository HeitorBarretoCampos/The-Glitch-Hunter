# The-Glitch-Hunter
um mini-game de sobrevivência cibernética onde você assume o papel de um Analista de Segurança em uma rede sob ataque. Seu objetivo é simples, mas crítico: identificar e neutralizar "Glitches" antes que eles corrompam o sistema. Navegue por puzzles de lógica, valide entradas maliciosas e suba no ranking global de protetores da rede
